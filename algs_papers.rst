The winners
-----------
- ART: Automatic Registration Toolbox. http://www.nitrc.org/projects/art/

- IRTK: Image Registration Toolkit. http://www.doc.ic.ac.uk/~dr/software/

- SyN: Symmetric Normalization. SyN is available in the ANTS toolkit; a new release with tutorial is available at http://www.picsl.upenn.edu/ANTS/

- FNIRT: FMRIB's Nonlinear Image Registration Tool. (Maybe same page as FLIRT)

- SPM5 DARTEL Toolbox: Diffeomorphic Anatomical Registration using Exponentiated Lie algebra. Unified Segmentation, and the DARTEL Toolbox are part of the SPM5 package and may be downloaded from http://www.fil.ion.ucl.ac.uk/spm/software/spm5/

-  JRD-fluid: Jensen-RÃ©nyi Divergence fluid. No page.


Non-winners
-----------

- Demons: Diffeomorphic Demons. It is implemented as part of the finite difference solver framework within the Insight Toolkit (ITK) and can be downloaded at http://hdl.handle.net/1926/510

- FLIRT: FMRIB's Linear Image Registration Tool (ONLY LINEAR ONE). http://www.fmrib.ox.ac.uk/fsl/

- AIR: Automated Image Registration. http://air.bmap.ucla.edu

- ROMEO: Robust multigrid elastic registration based on optical flow. No page.

- ANIMAL: Automatic Nonlinear Image Matching and Anatomical Labeling. http://www.bic.mni.mcgill.ca/users/louis/MNI_ANIMAL_home/readme/
